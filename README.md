## Running the program

### Building

*Make sure CMake is installed*

`brew install cmake` if on MacOS


```bash
mkdir build

cd build

cmake ..

cmake --build .
```

### Running

*Make sure you are in the root of the project (P1) and have built the program using steps above.*

```
build/main
```
